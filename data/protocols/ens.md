### ENS
- Name expiry: parent expiry must be >= child expiry
- Wrapped names use ERC1155 — fuses control permissions
- unwrapETH2LD should use safeTransferFrom not transferFrom
- Registrar controller can create/renew — access control critical
- DNSSEC verification chain must validate each link
